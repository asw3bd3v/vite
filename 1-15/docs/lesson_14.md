# ESLint

Внедрим статический анализатор кода ESLint.

Т.к. будем выполнять проверку typescript файлов, то воспользуемся инструментом typescript-eslint. Данный плагин обеспечит поддержку ESLint в typescript файлах.

Процесс установки будет отличаться от действий автора.

Запускаем следующую команду и отвечаем на вопросы.

```
npm init @eslint/config
```

После этого будет сгенерирован файл .eslintrc.json.

Теперь запускаем анализатор с помощью команды

```
npx eslint .
```

Также создаем файл .eslintignore где указываем те файлы, которые нужно игнорировать.

```
*.js
```

Если же мы хотим настроить полную интеграцию ESLint со сборщиком, то воспользуемся плагином vite-plugin-checker.

```js
// vite.config.js
plugins: [
	checker({
		eslint: {
			lintCommand: 'eslint "./**/*.{ts,tsx}"',
		},
	}),
];
```
